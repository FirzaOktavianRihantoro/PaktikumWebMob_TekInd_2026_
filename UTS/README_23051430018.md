# Deskripsi Proyek

Aplikasi ini dibuat menggunakan React.js dan Bootstrap untuk membantu manager produksi dalam:

- Menginput laporan produksi harian
- Menghitung netto dan yield otomatis
- Menyimpan data ke LocalStorage
- Menampilkan dashboard KPI produksi
- Mengelola riwayat data produksi

---

# Teknologi yang Digunakan

- React.js
- React Router DOM
- Bootstrap 5
- LocalStorage Browser
- Vite
